dayoff-API

Api untuk mengecek hari libur dan cuti bersama

Format Response

{
    "tanggal": "2023-01-1",
    "tanggal_display": "Minggu, 1 Januari 2023",
    "keterangan": "Tahun Baru 2023",
    "is_cuti": false
}

Cara Akses API

  • Mendapatkan data hari libur tahun sekarang /api
  • Mendapatkan data hari libur pada tahun 2020 /api?year=2020
  • Mendapatkan data hari libur di bulan 5 pada tahun saat ini /api?month=5
  • Mendapatkan data hari libur di bulan 5 pada tahun 2023 /api?month=5&year=2023

Cara Kerja API ini?

API ini mengambil data dari tanggalan.com dengan cara di scraping setiap 1 bulan sekali menggunakan github action.
